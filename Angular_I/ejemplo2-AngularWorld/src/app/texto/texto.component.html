QUÉ ES ANGULAR ❓

Basicamente es un FrameWork, de Js que permite crear parginas Single Page Application Eso significa que en Angular nunca abrira otra pagina, Todo será indexado en una misma vista. Esto hace que sea una forma mas elegante, simple y Proactiva de desarrollo Web
